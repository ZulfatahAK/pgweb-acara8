# Web GIS Kabupaten Sleman

Proyek ini adalah aplikasi Web GIS sederhana untuk menampilkan data kecamatan di Kabupaten Sleman. Aplikasi ini memungkinkan pengguna untuk melihat, menambah, mengedit, dan menghapus data kecamatan.

## Fitur

*   Tampilan peta interaktif menggunakan Leaflet.js
*   Tabel data kecamatan yang terintegrasi dengan peta
*   Fungsi CRUD (Create, Read, Update, Delete) untuk data kecamatan
*   Tampilan responsif menggunakan Bootstrap

## Cara Penggunaan

1.  Pastikan Anda memiliki web server (misalnya XAMPP) dengan PHP dan MySQL.
2.  Buat database dengan nama `latihan_8`.
3.  Impor file `data_kecamatan.sql` ke dalam database `latihan_8`.
4.  Letakkan semua file proyek di dalam direktori web server Anda (misalnya `htdocs` di XAMPP).
5.  Buka browser dan akses aplikasi melalui `http://localhost/nama_folder_proyek`.
